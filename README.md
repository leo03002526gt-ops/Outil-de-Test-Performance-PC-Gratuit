# Outil Test Performance PC Gratuit

Ce script Python permet de **tester les performances de votre PC** (CPU et RAM).

## Installation
1. Installer Python (>=3.8)
2. Installer psutil : `pip install psutil`

## Usage
`python pc_performance_test.py`
